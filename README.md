## **Game Controls**
- **B:** View the book of recipes (click to turn pages)
- **A/D:** Walk left/right
- **Space:** Jump
- **F:** Fetch ingredients from resource spots
- **E:** Pick up/drop ingredients or place/remove them from cookware
- **Enter:** Cook
